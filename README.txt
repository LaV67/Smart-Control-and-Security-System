Description:
Smart system security and fire alarm and ACS (access control system) controlled via Ethernet. Based on PIC18F97j60.

Main hardware capability:
1.Two RFID-readers.
2.Ten fire/security sensor inputs.
3.1-Wire port for temperature sensors.
4.Ethernet port.
5.RS-485 port.
6.Six independent relays.
7.Flash memory for 40k events and ID-codes.
